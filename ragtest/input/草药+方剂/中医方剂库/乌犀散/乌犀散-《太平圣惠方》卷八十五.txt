```markdown
# 乌犀散
    认证专家：中国中医科学院
    组成
    - 乌犀角屑 1分  
    - 独角仙 3枚（微炙，去翅足）  
    - 驴胎耳 1分（烧灰）  
    - 雀儿饭瓮 5枚  
    - 干蟾 1分（烧灰）  
    - 白僵蚕 2分（微炒）  
    - 朱砂 1分（细研）  
    - 雄黄 1分（细研）  
    - 丁香 1分  
    - 蚕纸 1张（出子者，烧）  
    - 麝香 1分（细研）  
    - 牛黄 1分（细研）  
    - 羌活 半两  
    - 青黛 1分（细研）  
    - 天竹黄 1分（细研）  
    主治
    小儿慢惊风，或发作时眼向上翻，手足抽搐。
    用法
    将上述药物研成细末，混合均匀。每次服用半钱，用温水调服，不拘时间。
    备考
    方中干蟾，《普济方》作“干蝎”。
    方源
    《太平圣惠方》卷八十五
```
