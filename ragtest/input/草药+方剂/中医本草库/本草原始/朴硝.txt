# 朴硝

生于益州山谷，有咸水之阳。采收时间无特定限制。用清水淋取汁液，一次煎煮而成，未经二次炼制，故称朴硝。又名硝石。朴硝是其本体名称；石者，是坚硬洁白之称；朴者，即未变化之意。因其芒消、英消皆由此产生，故称消石朴。其末消即今俗称为马牙消者是也。李时珍说：此物遇水即溶。又能化解各种物质，故称消。生于盐卤之地，状似末盐，故有盐消之名。凡牛马等皮革须用此物处理熟化，故俗称皮消。煎炼入盆凝结，底部粗朴者为朴硝，上部有锋芒者为芒消。形体大于芒消，与马牙消无异者为马牙消。置于风中吹去水气，则轻白如粉，为风化消。用朴硝十斤，水一桶，同入锅内溶解，去除表面油腻，将水用细布过滤去渣，用萝卜十斤、冬瓜五斤、豆腐三斤，均切厚片，同消水入锅内煮六七沸，捞出萝卜等物，再去除油腻，再过滤，令渣去除干净，放入瓦盆内，置于星月之下，自然生出消牙。取出放于桌上，任其风干，将原水再煎一沸，入瓦盆，令其再生消牙。如此多次，以水中不再生消牙为度。如前风干，用罐子装盛，压实泥封，周围用碎炭围住，防止火气逸散，煎炼一昼夜。待冷取出，置于洁净地面，以新瓦盆覆盖，以去除火毒。后研为末。每斤加生熟甘草面各一两，混合均匀，为玄明粉。

# 朴硝【性味】苦，寒，无毒。

【主治】百病，除寒热邪气，驱逐六腑积聚，结固留癖，能化解七十二种结石。炼制后服用，可轻身成仙。○胃中饮食热结，破留血闭绝，停痰痞满，推陈致新。○疗热胀，养胃消谷。○治腹胀，大小便不通，女子月经不通。○通泄五脏百百病及癥结，治天行热疾，头痛，消肿毒，排脓，润毛发。

# 芒消【性味】辛、苦，大寒，无毒。

【主治】五脏积聚，久热胃闭，除邪气，破留血，腹中痰实结搏，通经脉，利大小便及月水，破五淋，推陈致新。○下瘰疬、黄疸病，时疾壅热，能散恶血，堕胎。敷漆疮。

# 马牙消【性味】甘，大寒，无毒。

【主治】除五脏积热伏气。○研末，点眼赤，去赤肿障翳涩泪痛，亦入点眼药中用。○功效与芒消相同。

# 风化消【主治】上焦风热，小儿惊热膈痰，清肺解暑。以人乳调和涂敷，可去眼睑赤肿，及头面暴热肿痛。煎黄连，点赤目。

# 生消类朴硝而小坚，不经煮炼而成。生于茂州西山岩石间，其形块大小不一，色青白，采收时间无特定限制。恶麦句姜。【性味】苦，大寒，无毒。

【主治】风热癫痫，小儿惊邪瘛疭，风眩头痛，肺壅耳聋，口疮喉痹咽塞，牙颔肿痛，目赤热痛，多眵泪。

# 玄明粉【性味】辛、甘，冷，无毒。【主治】心热燥烦，并五脏宿滞癥结。○明目，退膈上虚热，消肿毒。

# 治风眼赤烂：明净皮硝一盏。水一碗煎化，露一夜，滤净澄清。朝夕洗目。三日其红即消，虽经数十年者亦愈也。

# 朴硝为君药。《别录》曰：苦、辛，大寒，炼白如银，能寒能热，能滑能涩，能辛、能咸、能酸。入地千年不变。权曰：有小毒，降也，阴也。

# 之才曰：石韦为之使；恶麦句姜；畏三棱。

# 芒消 权曰：有小毒。元素曰：气薄味厚，沉而降，阴也。一去实热，二涤肠中宿垢，三破坚积。孕妇惟三、四月及七、八月不可用，余用无妨。

# 骨蒸热病，芒消末，水服方寸匕，日二服，良。

# 《千金方》：治漆疮，用芒消浓涂之，干即再涂，二三次效。

# 《梅师方》：治火月毒，水调芒消涂之。《子母秘录》：治小儿赤游行于身体上下，延至心胸即死。以芒消内汤中，取浓汁以拭丹上，大效。

# 治食物过多不消，遂成痞膈，用马牙消一两碎之，吴茱萸陈者半升，煎汁投消，乘热服。良久未转，更进一服，立愈。

# 风化消 时珍曰：甘缓轻浮，故治上焦心肺痰热而不泄利。○玄明粉沉也，阴也。忌苦参。脾胃虚冷及阴虚火动者服之，是速其咎矣。

# 《伤寒蕴要》：治伤寒发狂者，用玄明粉二钱，朱砂二钱，为末，冷水调服。
