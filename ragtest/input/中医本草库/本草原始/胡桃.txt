# 胡桃

胡桃树高约一丈，春季初生叶片，长度约四五寸，两两相对。三月开花如栗花，花穗呈苍黄色。果实至秋季成熟如青桃。此种原产于羌胡地区，汉代张骞出使西域，始将种子带回。种植于秦中地区，逐渐传播至东方，因此得名胡桃，又名羌桃。将青皮肉沤烂后取核为果，故民间称其为核桃。

核仁【气味】甘，平，温，无毒。
【主治】食用后可使人健壮，滋润肌肤，使胡须和头发变黑。过量食用有利小便，可治疗五痔。捣碎与胡粉混合，可拔除白须发，内服可使生黑发。烧存性后与松脂研磨，可敷治瘰疬疮。○食用后可增进食欲，通畅血脉，使骨骼肌肉细腻。○治疗损伤和石淋，与破故纸同用蜜丸服用，可补益下焦。○补气养血，润燥化痰，益命门，利三焦，温肺润肠。可治疗虚寒喘嗽，腰脚沉重疼痛，心腹疝痛，血痢肠风，散肿毒，发痘疮，解铜毒。

胡桃，宋《开宝》。

# 油胡桃【气味】辛，热，有毒。
【主治】杀虫攻毒，治疗痈肿、疠风、疥癣、杨梅疮、白秃疮等。滋润须发。

# 胡桃青皮【气味】苦、涩，无毒。【主治】染须及染布皆可使颜色变黑。
仙方取胡桃外层青皮压榨取油，与詹糖香料混合涂抹毛发，颜色如漆。

# 胡桃树皮【主治】止水痢。春月砍取树皮汁液洗头可使头发变黑，煎水可染褐。

# 胡桃核壳【主治】烧存性，入治下血崩的中药中。

# 多食动痰饮，令人恶心、吐水、吐痰食。生痰，动风，动肾火。脱人眉毛。同酒食用，多令人咯血。

# 戏术：嚼钱如粉：预置胡桃肉一块于口中，将铜钱嚼之即碎。
误吞铜钱，多食胡桃可使其自行排出。胡桃与铜钱共嚼成粉可验证。
